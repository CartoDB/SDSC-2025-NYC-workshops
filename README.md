# SDSC-2025-NYC-workshops

# Heading 1  
Use `#` for the largest heading  

## Heading 2  
Use `##` for a sub-heading  

### Heading 3  
Use `###` for a smaller sub-heading  

#### Heading 4  
Use `####` for even smaller headings  

---

## Text Styles  

- **Bold** → `**Bold**`  
- *Italic* → `*Italic*`  
- ~~Strikethrough~~ → `~~Strikethrough~~`  
- `Inline code` → `` `Inline code` ``  

Blockquote:  
> Use `>` for blockquotes  

---

## Lists  

Unordered list (use `-`, `*`, or `+`):  
- Item 1  
- Item 2  
  - Nested Item  

Ordered list (use `1.`):  
1. Step one  
2. Step two  
   1. Sub-step  

Task list:  
- [x] Done  
- [ ] To do  

---

## Links & Images  

Link → `[link text](https://example.com)`  
- Example: [GitHub](https://github.com)  

Image → `![alt text](https://via.placeholder.com/150)`  
- Example:  
  ![Placeholder image](https://via.placeholder.com/150)
